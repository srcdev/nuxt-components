# MCP Component Generator Script

This script helps automatically generate MCP component documentation files.

## Usage

```bash
# Generate MCP file for a new component
node .mcp/generate-component-mcp.js ComponentName forms "Brief description"
```

## Manual Creation

For manual creation, follow the template in `/components/README.md`

## Auto-Discovery

Future enhancement: Script could scan component directories and auto-generate basic MCP files for undocumented components.
